# 🎮 Rock Paper Scissors Game

A simple and interactive **Rock Paper Scissors** game built using **Python**. The player competes against the computer, which makes random choices each round.

## 📌 Features

* User vs Computer gameplay
* Random computer choice using Python's `random` module
* Input validation
* Win, Lose, and Tie detection unctionality
* Option to play multiple rounds

## 🛠️ Technologies Used

* Python 3
* Random Module
